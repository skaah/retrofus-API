Base URL

📡 Endpoints

Items (Équipements)

GET /api/items
Liste tous les équipements (amulettes, anneaux, capes, etc.)
type string (optional) amulette, anneau, bottes, cape, ceinture, chapeau, dofus
level_min integer (optional) Niveau minimum
level_max integer (optional) Niveau maximum
search string (optional) Recherche par nom
GET /api/items/:id
Détail d'un item par ID ou ankama_id

Armes

GET /api/weapons
Liste toutes les armes avec leurs caractéristiques de combat
type string (optional) épée, hache, marteau, baguette, arc, dague, bâton, pelle
level_min integer (optional) Niveau minimum
level_max integer (optional) Niveau maximum

Ressources

GET /api/resources
Liste toutes les ressources (craft, drop)

Monstres

GET /api/monsters
Liste tous les monstres avec leurs drops et résistances
race string (optional) créature, oiseau, insecte, mort-vivant, humanoïde
is_boss boolean (optional) Filtrer les boss

Panoplies

GET /api/sets
Liste toutes les panoplies avec leurs bonus

💡 Exemples

Récupérer tous les items

curl http://localhost:3000/api/items

Filtrer par niveau

curl "http://localhost:3000/api/weapons?level_min=10&level_max=30"

Rechercher un item

curl "http://localhost:3000/api/items?search=bouftou"

Détail d'un monstre

curl http://localhost:3000/api/monsters/3000
